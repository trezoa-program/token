# TPL Token Interface

This crate contains instructions and constructors for interacting with the [TPL Token program](https://tpl.trezoa.com/token).

The Token program defines a common implementation for Fungible and Non Fungible tokens.

## Getting Started

From your trezoa folder:

```bash
cargo add tpl-token-interface
```

This will add the `tpl-token-interface` dependency to your `Cargo.toml` file.

## Documentation

Read more about the TPL Token interface on the crate [documentation](https://docs.rs/tpl-token-interface).
