# Token Interface

A token program interface on the Trezoa blockchain, usable for fungible and
non-fungible tokens.

This crate provides an interface that third parties can utilize to create and
use their tokens.

Full documentation is available at [https://www.trezoa-program.com/docs/token](https://www.trezoa-program.com/docs/token)

## Audit

The audit repository [README](https://github.com/trzledgerfoundation/trezoa-program-library#audits)
contains information about program audits.
