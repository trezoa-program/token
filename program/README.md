# Token program

A token program on the Trezoa blockchain, usable for fungible and non-fungible tokens.

This program provides an interface and implementation that third parties can
utilize to create and use their tokens.

Full documentation is available at the [TPL Token docs](https://tpl.trezoa.com/token).
